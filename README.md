# IG Technologies — Landing Page

Marketing landing page for IG Technologies, an AI-powered cybersecurity
company offering threat detection, managed detection & response (MDR),
penetration testing, cloud security posture management, identity threat
analytics, and security awareness training.

Built with [Next.js](https://nextjs.org) (App Router) and Tailwind CSS 4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Structure

- `app/page.tsx` — composes the landing page from the sections below
- `app/components/` — Navbar, Hero, Services, Stats, Platform, Approach,
  Industries, Testimonials, CTA, and Footer sections
- `app/globals.css` — dark theme design tokens (colors, grid background,
  gradient text, card styles)

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run start` — serve the production build
- `npm run lint` — run ESLint
